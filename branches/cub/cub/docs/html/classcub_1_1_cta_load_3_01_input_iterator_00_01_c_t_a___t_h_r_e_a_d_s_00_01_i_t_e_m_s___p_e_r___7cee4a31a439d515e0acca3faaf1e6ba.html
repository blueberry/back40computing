(inferred) Integer counting type

Load tile.

Specialized for native pointer types (attempts vectorization)

The aggregate set of items is assumed to be ordered across threads in "blocked" fashion, i.e., each thread owns an array of logically-consecutive items (and consecutive thread ranks own logically-consecutive arrays).(in) Offset in ptr at which to load the tile
